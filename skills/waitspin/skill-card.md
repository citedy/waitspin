# WaitSpin

WaitSpin helps agents use the public WaitSpin CLI safely: request and verify email OTP keys, create advertiser campaigns, open checkout, install supported earning surfaces, inspect wallet readiness, and verify package/source truth before making launch claims.

Install from the public source:

```bash
npx skills add citedy/waitspin --skill waitspin -g -y
```

Primary source: `https://github.com/citedy/waitspin`
