# Data Boundary

WaitSpin public clients fetch and display sponsored wait-state lines. They do
not inspect or transmit developer work. See TRUST.md for the never-sent list
and operational payloads.

Operational telemetry means serve, impression, wallet, accounting, rate-limit,
fraud-control, and audit events needed to run the marketplace. WaitSpin does
not add a separate analytics telemetry stream in the public clients.
