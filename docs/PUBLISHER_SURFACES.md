# Publisher Surfaces

- VS Code: status-bar-fallback; SecretStorage plus Activity Bar/status-bar fallback.
- Cursor Editor Mode: status-bar-fallback; same VS Code-compatible Marketplace extension via cursor --install-extension waitspin.waitspin-vscode.
- Devin Desktop: status-bar-fallback; Open VSX-published VS Code-compatible extension via devin-desktop --install-extension waitspin.waitspin-vscode when the desktop CLI is on PATH.
- Claude Code: claude-code; statusLine.command with compose-existing support.
- MiMo Code: mimocode; managed runtime and bash hook.
- OpenCode: opencode; plugin under ~/.config/opencode/plugins and tui.json entry.
- Grok Code CLI: grok; managed text-asset footer patch with hash-backed backup/restore.
- Antigravity CLI: antigravity; statusLine.command with compose-existing support.
- GitHub Copilot CLI: copilot; statusLine.command with compose-existing support.

Cline, Kimi, MMX, and other experimental native CLI targets are not public targets unless separately promoted.
